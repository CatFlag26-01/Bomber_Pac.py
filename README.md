Este programa fue hecho con Python y distintos módulos:

sys - Que se utilizara como la librería para manejar argumentos y salidas
Pygame - Es un conjunto de módulos de Python de código abierto, diseñado para el desarrollo de videojuegos 2D y aplicaciones multimedia. Basado en la biblioteca SDL (Simple DirectMedia Layer), permite gestionar gráficos, sonido, teclado y ratón
Random - "Random" es un anglicismo que significa aleatorio, al azar, inesperado o sin un orden establecido. Coloquialmente, se usa para describir algo extraño, absurdo, o una persona desconocida/cualquiera.
Mecanicas de Bomberman - Las mecánicas de Bomberman se definen como un juego de acción y estrategia basado en laberintos de cuadrícula, donde el objetivo principal es eliminar enemigos y obstáculos colocando bombas que explotan en forma de cruz. El núcleo del juego implica una gestión táctica del espacio y el tiempo
